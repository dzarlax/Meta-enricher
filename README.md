# MetaEnricher

> AI-powered photo metadata enrichment for macOS and Windows.

---

## The problem

You bought a camera. You come home with a memory card full of photos. Now what?

Most beginner photographers face the same wall: the camera gives you a `DCIM` folder with hundreds of cryptically named files (`DSC_0047.JPG`), no structure, no context, no metadata. You don't know where to put them, how to name folders, how to separate RAW from JPEG, or how to keep track of what was shot when and where. And even if you figure out the folder structure, every photo still needs a title, description, keywords, and copyright before you can publish or sell it.

The usual answers don't always fit:

- **Apple Photos** locks your library inside its own database — great if you live entirely in that ecosystem, but frustrating if you want to edit in external apps, manage files directly, or keep your library as plain folders on disk
- **Lightroom** solves the organization problem well, but it's a $10/month subscription just to get started
- **Capture One, Photo Mechanic** — powerful, but overkill and expensive for someone just starting out

MetaEnricher is for everyone who wants to **own their files as plain folders**, skip the expensive subscriptions, and still get a professional-grade library structure and metadata — with the help of AI.

It gives you **a clear, opinionated folder structure** for your entire photo library, and it uses **a vision AI model** to automatically enrich every photo with meaningful metadata — title, caption, keywords, location, copyright — in seconds.

You choose where the AI runs: **locally on your machine** for full privacy, or **in the cloud** if you don't have a GPU.

---

## How it works

1. **Import your photos** — bring photos in from an SD card / camera (auto-detected via DCIM) or from any folder already on your PC; files are copied to your library and organized automatically by date
2. **Review your sessions** — each shooting day appears as a session in the sidebar; browse your edited exports
3. **Enrich with AI** — one click sends photos to a vision AI model, which generates title, description, keywords, and location for each image
4. **Metadata is written to the files** — EXIF, IPTC, and XMP tags are saved directly into your JPEGs, ready for Lightroom, stock platforms, or wherever you publish

---

## Features

- **AI metadata generation** — title, caption, keywords, and location for each photo, generated by a vision model (e.g. Qwen2.5-VL or Qwen3-VL)
- **Local or cloud AI** — run [Ollama](https://ollama.com) on your own machine for full privacy, or use Ollama Cloud if you don't have a GPU
- **Session-based workflow** — library organized by shooting sessions, discovered automatically from your folder structure
- **EXIF / IPTC / XMP writing** — metadata written directly to files, compatible with Lightroom, Capture One, Photo Mechanic, and stock platforms
- **Smart location** — GPS coordinates reverse-geocoded via OpenStreetMap; AI fills in location when GPS is absent; session notes provide extra context
- **Bulk enrichment** — enrich one photo or an entire session in one click
- **Flexible import** — pull photos from an SD card, camera, or phone via DCIM auto-detect, or point at any folder already on your PC, network share, or mounted device

## Install

### Windows

Grab the latest release from [Releases](https://github.com/Dzarlax-AI/Meta-enricher/releases):

- **`MetaEnricher-vX.Y.Z-portable-x64.zip`** — single-file portable .exe, no installer, just unzip and run
- **`MetaEnricher_X.Y.Z.0_x64_arm64.msixbundle`** — sideload bundle for x64 + ARM64 (requires importing the bundled certificate the first time, see the release notes)

Microsoft Store version is currently going through certification review.

Requires Windows 10 (1809) or later.

### macOS

Open `macOS/MetaEnricher/MetaEnricher.xcodeproj` in Xcode and ⌘R to build & run, or grab a build from a future release. Requires macOS 26 or later.

### AI backend

MetaEnricher needs a vision model to enrich photos. Two options:

**Local (private, free, requires GPU):**
```bash
# https://ollama.com — install Ollama, then pull a vision model
ollama pull qwen2.5vl
ollama serve
```

**Cloud (works without GPU):** sign up for an [Ollama Cloud](https://ollama.com) account, enter your API key in MetaEnricher settings.

## Library structure

Instead of leaving you with a chaotic dump of `DSC_0001.JPG` files, MetaEnricher organizes your library by year and shooting session. The structure is created automatically on import — you never have to think about it:

```
CAMERA_ROOT/
  2024/
    2024-07-04 Fireworks/
      Edited export/   ← what MetaEnricher browses and enriches
      JPEG/            ← original JPEGs
      RAW/             ← original RAW files
```

## Setup

1. Launch MetaEnricher
2. Complete the onboarding wizard — pick your library root folder and configure the AI backend (local Ollama or Ollama Cloud)
3. Select a session from the sidebar
4. Click **Enrich** on any photo or use **Enrich All** for the whole session

Default creator name and copyright notice can be set in **Settings** and will be pre-filled on every enriched photo.

---

## Privacy

MetaEnricher stores no data remotely. By default it talks only to your **local** Ollama instance and to **OpenStreetMap's Nominatim** for reverse geocoding (coordinates → city/country name, no images sent).

If you opt into **Ollama Cloud** mode, photo thumbnails are sent to Ollama's servers for processing — Ollama's privacy policy applies. Your API key is stored locally only.

See [PRIVACY_POLICY.md](PRIVACY_POLICY.md) for the full policy.
